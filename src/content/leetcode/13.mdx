---
title: 13. Roman to Integer
description: Given a roman numeral, convert it to an integer.
difficulty: Easy
tags: ["Math", "String"]
---

## Solution
```py
class Solution:
    def romanToInt(self, s: str) -> int:
        roman = {
            'I': 1,
            'V': 5,
            'X': 10,
            'L': 50,
            'C': 100,
            'D': 500,
            'M': 1000
        }
        ans = 0
        prev_value = 0

        for c in s:
            curr_value = roman[c]
            if curr_value > prev_value:
                ans += curr_value - 2 * prev_value
            else:
                ans += curr_value
            prev_value = curr_value
        
        return ans
```
